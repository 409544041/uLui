基于ugui组件封装，高级控件库uLui
====
![](picture/photo1.png)
特性
-------  
### 组件LWidget
1.新手引导层，带穿透孔<br>
2.可复用的富文本<br>
3.摇杆控件<br>
4.网格容器<br>
5.网格翻页容器<br>
6.序列帧控件<br>
7.翻页容器<br>
8.长按按钮<br>
9.复用列表<br>

### 窗体LWindow
1.自动管理窗体层次<br>
2.窗体回收和缓存<br>
3.窗体逻辑跟控件解耦<br>

运行
-------  
双击运行 Assets/Resources/Scenes/first 场景即可
